# A-Frame Project

This folder is set up with Next.js, React, TypeScript, and Zustand.

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000`.

## Included Stack

- Next.js App Router
- React
- TypeScript
- Zustand
- Tailwind CSS v4

## Example State Setup

- Store: `stores/use-counter-store.ts`
- Client component: `components/counter-panel.tsx`
- Page: `app/page.tsx`

## Scripts

```bash
npm run dev
npm run build
npm run lint
```
